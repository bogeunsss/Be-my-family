Be My Family
=========
### 유기견이 가족을 찾을 수 있도록 도와주는 Web Service

  

  
개발환경
=========
__SpringBoot, Maven, JPA, MariaDB, Docker, VUE.JS, AWS__

  

페이지 구성
=========
+ 메인 페이지  
	+ 유기견 보호소 게시판 미리보기  
	+ 실종/보호/목격 게시판 미리보기  
	+ 입양 후기  
+ 유기견 보호소 게시판  
	+ 카테고리 별 검색  
	+ 보호중인 유기견 목록(상세보기)  
		+ 관심목록 등록  
		+ 입양신청  
+ 실종/보호/목격 게시판  
	+ 태그 별 검색  
	+ 게시판 목록(상세보기)  
		+ 댓글  
		+ 공유  
	+ 게시글 등록  
+ 입양후기(구현중)
